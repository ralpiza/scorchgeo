# Installation

## Stable release

To install scorchgeo, run this command in your terminal:

```
pip install scorchgeo
```

This is the preferred method to install scorchgeo, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

To install scorchgeo from sources, run this command in your terminal:

```
pip install git+https://github.com/ralpiza/scorchgeo
```
